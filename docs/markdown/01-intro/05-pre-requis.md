# Pre-requis pour développer en local

Pour les labs, nous avons besoin d'avoir nodejs
```shell
https://nodejs.org/en
```
<!-- .element: class="big-code" -->

Exécutez la commande npm pour installer la CLI

```shell
npm install -g firebase-tools
firebase login
```
